# EntryFpslAction

Indicates whether shares are being allocated or deallocated


## Values

| Name                                   | Value                                  |
| -------------------------------------- | -------------------------------------- |
| `EntryFpslActionFpslActionUnspecified` | FPSL_ACTION_UNSPECIFIED                |
| `EntryFpslActionAllocate`              | ALLOCATE                               |
| `EntryFpslActionDeallocate`            | DEALLOCATE                             |
