# AccountState

Indicates if the account is `OPEN`, `PENDING`, or `CLOSED`


## Values

| Name                           | Value                          |
| ------------------------------ | ------------------------------ |
| `AccountStateStateUnspecified` | STATE_UNSPECIFIED              |
| `AccountStatePending`          | PENDING                        |
| `AccountStateOpen`             | OPEN                           |
| `AccountStateClosed`           | CLOSED                         |
