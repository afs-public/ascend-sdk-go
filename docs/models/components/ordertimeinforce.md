# OrderTimeInForce

For Equities: Either "DAY" or "GOOD_TILL_DATE" are allowed. For Mutual Funds: Only "DAY" is allowed. For Fixed Income: Only "DAY" is allowed.


## Values

| Name                           | Value                          |
| ------------------------------ | ------------------------------ |
| `OrderTimeInForceDay`          | DAY                            |
| `OrderTimeInForceGoodTillDate` | GOOD_TILL_DATE                 |
