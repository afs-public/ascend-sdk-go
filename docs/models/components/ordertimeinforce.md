# OrderTimeInForce

For Equities: Either "DAY" or "GOOD_TILL_DATE" are allowed. For Mutual Funds: Only "DAY" is allowed. For Fixed Income: Only "DAY" is allowed. For Event Contracts: Either "DAY", "GOOD_TILL_DATE", "GOOD_TILL_CANCELED", "IMMEDIATE_OR_CANCEL", or "FILL_OR_KILL" are allowed.


## Values

| Name                                | Value                               |
| ----------------------------------- | ----------------------------------- |
| `OrderTimeInForceDay`               | DAY                                 |
| `OrderTimeInForceGoodTillDate`      | GOOD_TILL_DATE                      |
| `OrderTimeInForceGoodTillCanceled`  | GOOD_TILL_CANCELED                  |
| `OrderTimeInForceImmediateOrCancel` | IMMEDIATE_OR_CANCEL                 |
| `OrderTimeInForceFillOrKill`        | FILL_OR_KILL                        |
