# OrderSettlementTarget

The desired target settlement state of the order. If unspecified, the order settlement state will default to "FILLED".


## Values

| Name                                                    | Value                                                   |
| ------------------------------------------------------- | ------------------------------------------------------- |
| `OrderSettlementTargetOrderSettlementTargetUnspecified` | ORDER_SETTLEMENT_TARGET_UNSPECIFIED                     |
| `OrderSettlementTargetBooked`                           | BOOKED                                                  |
| `OrderSettlementTargetFilled`                           | FILLED                                                  |
