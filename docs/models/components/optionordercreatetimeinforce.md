# OptionOrderCreateTimeInForce

Must be the value "DAY". Regulatory requirements dictate the system captures the intended time_in_force, which is why this a mandatory field.

| ---------------------------------------------------- | ---------------------------------------------------- |
| `OptionOrderCreateTimeInForceTimeInForceUnspecified` | TIME_IN_FORCE_UNSPECIFIED                            |
| `OptionOrderCreateTimeInForceDay`                    | DAY                                                  |
