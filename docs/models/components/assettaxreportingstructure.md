# AssetTaxReportingStructure

Indicates whether the investment generates IRS Form 1099 or Schedule K-1 for tax reporting. Critical for year-end tax processing.


## Values

| Name                                                         | Value                                                        |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| `AssetTaxReportingStructureTaxReportingStructureUnspecified` | TAX_REPORTING_STRUCTURE_UNSPECIFIED                          |
| `AssetTaxReportingStructureForm1099`                         | FORM_1099                                                    |
| `AssetTaxReportingStructureFormK1`                           | FORM_K1                                                      |
