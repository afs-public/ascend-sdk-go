# TradingFeeType

The type of fee being specified. Only the type of "BROKER_FEE" is supported.


## Values

| Name                               | Value                              |
| ---------------------------------- | ---------------------------------- |
| `TradingFeeTypeFeeTypeUnspecified` | FEE_TYPE_UNSPECIFIED               |
| `TradingFeeTypeBrokerFee`          | BROKER_FEE                         |
| `TradingFeeTypeContractFee`        | CONTRACT_FEE                       |
