Indicates how frequently an account can redeem the asset.

| `RedemptionFrequencyFrequencyUnspecified` | FREQUENCY_UNSPECIFIED                     |
| `RedemptionFrequencyDaily`                | DAILY                                     |
| `RedemptionFrequencyWeekly`               | WEEKLY                                    |
| `RedemptionFrequencyBiWeekly`             | BI_WEEKLY                                 |
| `RedemptionFrequencyMonthly`              | MONTHLY                                   |
| `RedemptionFrequencyQuarterly`            | QUARTERLY                                 |
| `RedemptionFrequencyBiAnnually`           | BI_ANNUALLY                               |
| `RedemptionFrequencyAnnually`             | ANNUALLY                                  |
