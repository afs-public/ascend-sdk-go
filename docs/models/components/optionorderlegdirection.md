# OptionOrderLegDirection

The computed direction for this leg. When `side` is BUY, `direction` will be DEBIT. When `side` is SELL, `direction` will be CREDIT.


## Values

| Name                                                | Value                                               |
| --------------------------------------------------- | --------------------------------------------------- |
| `OptionOrderLegDirectionDebitCreditTypeUnspecified` | DEBIT_CREDIT_TYPE_UNSPECIFIED                       |
| `OptionOrderLegDirectionDebit`                      | DEBIT                                               |
| `OptionOrderLegDirectionCredit`                     | CREDIT                                              |
