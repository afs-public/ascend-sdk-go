# RecordState

Indicates the current state of the asset record.  The record state identifies which record types the asset allows.


## Values

| Name                                | Value                               |
| ----------------------------------- | ----------------------------------- |
| `RecordStateRecordStateUnspecified` | RECORD_STATE_UNSPECIFIED            |
| `RecordStateActive`                 | ACTIVE                              |
| `RecordStateFrozen`                 | FROZEN                              |
| `RecordStateInactive`               | INACTIVE                            |
