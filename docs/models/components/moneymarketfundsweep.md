# MoneyMarketFundSweep

Option to auto-enroll in Money Market Fund Sweep; defaults to MONEY_MARKET_FUND_SWEEP_ENROLL


## Values

| Name                                                            | Value                                                           |
| --------------------------------------------------------------- | --------------------------------------------------------------- |
| `MoneyMarketFundSweepAutoEnrollMoneyMarketFundSweepUnspecified` | AUTO_ENROLL_MONEY_MARKET_FUND_SWEEP_UNSPECIFIED                 |
| `MoneyMarketFundSweepMoneyMarketFundSweepEnroll`                | MONEY_MARKET_FUND_SWEEP_ENROLL                                  |
| `MoneyMarketFundSweepMoneyMarketFundSweepDecline`               | MONEY_MARKET_FUND_SWEEP_DECLINE                                 |
