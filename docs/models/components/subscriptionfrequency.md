# SubscriptionFrequency

Indicates how frequently the asset will accept subscriptions (that is, how often an account is allowed to buy the asset).


## Values

| Name                                        | Value                                       |
| ------------------------------------------- | ------------------------------------------- |
| `SubscriptionFrequencyFrequencyUnspecified` | FREQUENCY_UNSPECIFIED                       |
| `SubscriptionFrequencyDaily`                | DAILY                                       |
| `SubscriptionFrequencyWeekly`               | WEEKLY                                      |
| `SubscriptionFrequencyBiWeekly`             | BI_WEEKLY                                   |
| `SubscriptionFrequencyMonthly`              | MONTHLY                                     |
| `SubscriptionFrequencyQuarterly`            | QUARTERLY                                   |
| `SubscriptionFrequencyBiAnnually`           | BI_ANNUALLY                                 |
| `SubscriptionFrequencyAnnually`             | ANNUALLY                                    |
