# FeeCreateType

The type of fee being specified. Only the type of "BROKER_FEE" is supported.


## Values

| Name                              | Value                             |
| --------------------------------- | --------------------------------- |
| `FeeCreateTypeFeeTypeUnspecified` | FEE_TYPE_UNSPECIFIED              |
| `FeeCreateTypeBrokerFee`          | BROKER_FEE                        |
| `FeeCreateTypeContractFee`        | CONTRACT_FEE                      |
