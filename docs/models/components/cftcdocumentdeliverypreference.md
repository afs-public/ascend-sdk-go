# CftcDocumentDeliveryPreference

Delivery method instruction for CFTC documents for a given Interested Party record; Not set for interested parties (no email collected); only applies to CFTC regulated accounts


## Values

| Name                                                          | Value                                                         |
| ------------------------------------------------------------- | ------------------------------------------------------------- |
| `CftcDocumentDeliveryPreferenceDeliveryPreferenceUnspecified` | DELIVERY_PREFERENCE_UNSPECIFIED                               |
| `CftcDocumentDeliveryPreferenceDigital`                       | DIGITAL                                                       |
| `CftcDocumentDeliveryPreferencePhysical`                      | PHYSICAL                                                      |
| `CftcDocumentDeliveryPreferenceSuppress`                      | SUPPRESS                                                      |
