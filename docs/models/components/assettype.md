# AssetType

The type of the asset in this order, which must be one of the following:
 EQUITY, MUTUAL_FUND, and FIXED_INCOME.


## Values

| Name                   | Value                  |
| ---------------------- | ---------------------- |
| `AssetTypeEquity`      | EQUITY                 |
| `AssetTypeFixedIncome` | FIXED_INCOME           |
| `AssetTypeMutualFund`  | MUTUAL_FUND            |
