# AlternativeOrderCreateIdentifierType

Identifies which parameter value is used to identify the asset in the order. The value for that parameter is used in the `identifier` field.


## Values

| Name                                                            | Value                                                           |
| --------------------------------------------------------------- | --------------------------------------------------------------- |
| `AlternativeOrderCreateIdentifierTypeIdentifierTypeUnspecified` | IDENTIFIER_TYPE_UNSPECIFIED                                     |
| `AlternativeOrderCreateIdentifierTypeAssetID`                   | ASSET_ID                                                        |
