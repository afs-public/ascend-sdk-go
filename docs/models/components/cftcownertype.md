# CftcOwnerType

Indicates the CFTC (Commodity Futures Trading Commission) owner type of the account. This enum only applies to accounts regulated by the CFTC


## Values

| Name                                    | Value                                   |
| --------------------------------------- | --------------------------------------- |
| `CftcOwnerTypeCftcOwnerTypeUnspecified` | CFTC_OWNER_TYPE_UNSPECIFIED             |
| `CftcOwnerTypeCustomer`                 | CUSTOMER                                |
| `CftcOwnerTypeProprietary`              | PROPRIETARY                             |
